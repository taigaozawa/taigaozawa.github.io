詳細は，N 予備校プログラミングコース公式ページ をご覧ください．

概要

この教材の目標

主に JavaScript を用いて，小規模な Web サービスを自分一人で

  • 企画
  • 開発
  • 運用
できるようになること．

主な内容

章立て：

  1. HTML, CSS, JavaScript を学ぶ．
  2. Linux 環境構築や Git/GitHub の使い方など，開発手法の基本を学ぶ．
  3. サーバサイドの JavaScript プラットフォームである Node.js を用いて Web アプリの開発を行う．その際，Hubot, Heroku, XSS/CSRF 対策などについて学ぶ．
  4. 上記で学んだことをもとに，実践的な開発手法を学ぶ．その際，Express.js, OAuth, PostgreSQL, テスト，CI などについて学ぶ．
