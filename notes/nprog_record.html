入門コース「プログラミング入門 Web アプリ」

概要

入門コースの目標は，主に JavaScript を用いて小規模な Web サービスを自分一人で

  • 企画
  • 開発
  • 運用
できるようになること．

主な内容は次のとおり：

  1. HTML, CSS, JavaScript（クライアントサイド）を学ぶ．
  2. Linux 環境構築や Git/GitHub の使い方など，開発手法の基本を学ぶ．
  3. サーバサイドの JavaScript プラットフォームである Node.js を用いて Web アプリの開発を行う．その際，Hubot, Heroku, XSS/CSRF 対策などについて学ぶ．
  4. 上記で学んだことをもとに，実践的な開発手法を学ぶ．その際，Express.js, OAuth, PostgreSQL, テスト，CI などについて学ぶ．

第１章 はじめよう -- Web プログラミング入門

1-01 Chrome / console / JavaScript

Google Chrome のインストールを確認し，Chrome のデベロッパーツールの console で JavaScript を実行．

  • console 上で alert(3+3)（計算式）を入力．
    => アラートダイアログに 6（計算結果）が表示される．
  • piyo（文字列）を表示させたいときは，シングルクオーテーション（'）でくくって alert('piyo') とする．

1-02 VS Code / エディタの補助機能

Visual Studio Code (VS Code) をインストールし，日本語化の拡張機能を適用．

VS Code の主な補助機能：

  • 単語の補完機能（インテリセンス）
  • コードを手軽に記述するための機能（Emmet）
  • 単語の色を変える機能（シンタックスハイライト）※ シンタックス … 構文

1-03 HTML タグ

さまざまな HTML タグを学習．

HTML ではたとえば， 

<a href="google.com">Google</a>
の <a></a> など，タグとタグに囲まれた部分を要素，href="google.com" などの部分を要素の属性という．
href 属性は，ハイパーリンクのリンク先を a タグに情報として与える．

  • h タグ … 見出し．header の略．
  • p タグ … 段落．paragraph の略．
  • br タグ … 改行．line break の略．
  • a タグ … ハイパーリンク．anchor（船のいかり，アンカー）の略．
  • img タグ … 画像．image の略．src 属性は source（画像のソース）の略．alt 属性は alternate（代替）の略．alt 属性で代替テキストを指定できる．
  • ul タグ … 順序なしリスト．unordered list の略．
  • ol タグ … 順序つきリスト．ordered list の略．
  • li タグ … リストの項目．list item の略．ul または ol の子要素とする必要がある．
  • ul>li*6 のように入力して tab キーを押すとテンプレートが入力される．
  • table タグ … 表（table）．
  • tr タグ … 表の 1 行．table row の略．
  • th タグ … 表の見出し．table header の略．
  • td タグ … 表のデータ．table data の略．
  • table>(tr>(th*2))+(tr>td*2)*2 のように入力して tab キーを押すと，次のようなテンプレートが入力される： 
    <table>
    <tr>
        <th></th>
        <th></th>
    </tr>
    <tr>
        <td></td>
        <td></td>
    </tr>
    <tr>
        <td></td>
        <td></td>
    </tr>
</table>
    見出し A 見出し B
    データ 1 データ 2
    データ 3 データ 4
