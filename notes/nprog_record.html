入門コース「プログラミング入門 Web アプリ」

概要

入門コースでは，主に JavaScript を用いて小規模な Web サービスを自分一人で

  • 企画
  • 開発
  • 運用
できるようになることが目標である．

主な内容は次のとおり：

  1. HTML, CSS, JavaScript（クライアントサイド）を学ぶ．
  2. Linux 環境構築や Git/GitHub の使い方など，開発手法の基本を学ぶ．
  3. サーバサイドの JavaScript プラットフォームである Node.js を用いて Web アプリの開発を行う．その際，Hubot, Heroku, XSS/CSRF 対策などについて学ぶ．
  4. 上記で学んだことをもとに，実践的な開発手法を学ぶ．その際，Express.js, OAuth, PostgreSQL, テスト，CI などについて学ぶ．

第１章 はじめよう -- Web プログラミング入門

1-01

Google Chrome のインストールを確認し，Chrome のデベロッパーツールの console で JavaScript を実行．

  • console 上で alert(hoge)（hoge は計算式）を入力．=> アラートダイアログに hoge（計算結果）が表示される．
  • piyo（文字列）を表示させたいときは，シングルクオーテーション（'）でくくって alert('piyo') とする．

1-02

Visual Studio Code (VS Code) をインストールし，日本語化の拡張機能を適用．

VS Code の主な補助機能：

  • 単語の補完機能（インテリセンス）
  • コードを手軽に記述するための機能（Emmet）
  • 単語の色を変える機能（シンタックスハイライト）※ シンタックス = 構文
